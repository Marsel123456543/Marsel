к любой команде можно дописать --help и выйдет описание как использовать эту команду 

 cd test   - переходим в папку test (если это папка и она существует)
 cd ..     - переход в родительскую папку
 cd -      - путь в котором мы были до этого 
 cd .      - переход в эту же папку (мы никуда не перейдем)
 cd        - переход в домашнюю папку (~)
 pwd       - показывает абсолютный путь до текущей папки 
 mkdir test - создание папки test
 mkdir test/new_dir - создание новой папки new_dir в уже существующей папке test

 touch test.txt - создание файла test.txt

 rm test,txt - это удаление файла test.txt 
 rm -rf test - удаление папки test со всем его содержимым
 
 cat test.txt - просмотр содержимого файла test.txt
 ls test - просмотр всех файлов и папок внутри папки test 
 ls -a - просмотр всех файлов и папок внутри папки test включая скрытые
 ls -l - просмотр всех файлов и папок с более подробной информацией 

 nano text.txt - открыть файл в редакторе 
 чтобы сохранить (ctrl s)
 чтобы выйти (ctrl x)
